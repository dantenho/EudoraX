

# AI Agents & n8n Integration Strategy

## Overview
EudoraX leverages a hybrid architecture combining browser-based local AI (Gemini Nano), cloud-based GenAI (Gemini 2.5), and external orchestration via n8n and LangChain.

---

## Core Agents

### 1. The "Creative Director" Agent
*   **Role**: High-level task planning and asset generation orchestration.
*   **Tools**: `ImageGenerator`, `VideoGenerator`, `VoiceSynthesis`.
*   **Input**: Vague concept (e.g., "Make a promo for a cyberpunk coffee shop").
*   **Strategy**:
    1.  **Decompose**: Split intent into visual, audio, and motion components.
    2.  **Draft**: Generate prompt candidates for each tool.
    3.  **Execute**: Call the respective EudoraX APIs.

---

## Specialized Development Agents


### 2. The "Refactoring" Agent
*   **Role**: Improve code maintainability and readability without altering behavior.
*   **Trigger**: Manual invocation on code blocks.
*   **System Prompt**:
    > "You are a Senior Software Architect. Refactor the following code to follow SOLID principles. 
    > Extract complex logic into helper functions. Ensure variable names are semantic. Do not change the output."
*   **Workflow**:
    1.  Receive Code Snippet.
    2.  Identify "Code Smells" (Long functions, tight coupling).
    3.  Apply refactoring patterns.
    4.  Output cleaner code.

### 3. The "Optimization" Agent
*   **Role**: Performance tuning for WebGL/WebGPU and JavaScript execution.
*   **Focus Areas**: 
    *   `ImageGenerator` render loops.
    *   React component re-renders.
    *   ONNX Runtime execution providers.
*   **Strategy**:
    *   Suggest `useMemo` / `useCallback` wrappers.
    *   Recommend Web Worker offloading for heavy tasks.
    *   Identify potential memory leaks in canvas contexts.

### 4. The "Debugger & Depuration" Agent
*   **Role**: Error analysis and fix suggestion.
*   **Input**: Error Stack Trace + Relevant Code Snippet.
*   **System Prompt**:
    > "Analyze the following stack trace. Identify the root cause (e.g., null pointer, race condition, type mismatch). 
    > Propose a specific code fix."
*   **Integration**:
    *   Parses `geminiService` API errors (e.g., 400 Bad Request).

### 5. The "Data Plotting" Agent
*   **Role**: Visualizing data generated by Python scripts or n8n workflows.
*   **Tools**: Chart.js / Recharts (Generated Code).
*   **Workflow**:
    1.  Input: JSON Array or CSV data.
    2.  Analyze data distribution and types.
    3.  Select best visualization (Line, Bar, Scatter).
    4.  Generate React component code to render the plot.
