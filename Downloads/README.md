
# EudoraX Downloads Folder

This directory stores hardware-accelerated model weights and ONNX binaries.

## Directories
- `models/onnx/`: Optimized ONNX runtimes for WebNN fallback.
- `weights/safetensors/`: VLLM/LoRA style adapters.

## Jules Hint
Use `wget` with Astral `uv` for concurrent weight acquisition during system provisioning.
